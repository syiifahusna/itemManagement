Item Management

This project originally intended for managing storage item and stocks, then it suddenly became a grocery store.

Getting Started

Requirements
  • JDK: JDK 17 or higher
  • Build Tool: Maven
  • IDE: Intellij or Eclipse
  • Database: Postgres
  • Web Browser: Any modern browser
  • Testing: Postman
Installation Steps

1. Clone the project:

git clone https://github.com/syiifahusna/itemManagement.git

2. Use master branch:

git checkout master

3. Create database:

CREATE DATABASE itemmanagement;

4. Configure database in application.properties:

spring.datasource.url=jdbc:postgresql://localhost:5432/itemmanagement
spring.datasource.username=yourusername
spring.datasource.password=yourpassword

5. Compile project:

mvn compile package

6. Run project:

mvn spring-boot:run

  • Account verification
  • Password recovery
  • Update user's profile
  • Item catalog
  • Add and remove items
  • Item search bar by name and price
  • Add and remove item from cart
  • Update cart item quantity
  • Display total cost
  • Order
  • Order history and status update
  • Order search bar by order date and status
  • Modify order status
  • Update mailing server details

APIs

APIs can only be used by Client users. Use client authentication to get JWT token.
Request Endpoint Description
POST /api/auth Get JWT token
GET /api/items List of items
GET /api/item/{id} Item info
GET /test/{state} Testing API
